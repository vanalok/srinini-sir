### Summary for Webpage Design

| Section | Key Display Fields |
|---|---|
| **Hero / Header** | `title`, `firstPublishedDate`, `lastPublishedDate`, `minutesToRead`, `language` |
| **Cover Media** | `media.wixMedia.image` or `media.embedMedia.thumbnail` |
| **Body** | `richContent.nodes` (paragraphs, headings, images, videos, lists) |
| **Excerpt / Preview** | `excerpt` |
| **Author** | `owner.name`, `owner.image.url` |
| **Categories / Tags** | `categories[].label`, `hashtags`, `tagIds` |
| **Engagement** | `metrics.views`, `metrics.likes`, `metrics.comments`, `commentingEnabled` |
| **Related Posts** | `relatedPosts[].title`, `relatedPosts[].url`, `relatedPosts[].media` |
| **SEO** | `seoData.tags`, `seoData.settings.keywords` |
| **Settings** | `featured`, `pinned`, `preview` (for conditional rendering) |
